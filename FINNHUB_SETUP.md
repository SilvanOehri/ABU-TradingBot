# Finnhub API Configuration for Vercel

## Setup Steps:

1. **Get FREE Finnhub API Key:**
   - Go to: https://finnhub.io/register
   - Create free account 
   - Copy your API key

2. **Add to Vercel Environment Variables:**
   - Go to your Vercel dashboard
   - Select your project  
   - Go to Settings → Environment Variables
   - Add: `FINNHUB_API_KEY` = `your_api_key_here`

3. **Benefits:**
   - ✅ 60 requests/minute (free)
   - ✅ Real Tesla, Bitcoin, Apple data
   - ✅ Works perfectly with Vercel
   - ✅ No rate limiting issues
   - ✅ Official API with documentation

## Without API Key:
The bot will use "demo" mode with realistic fallback data based on current market prices.

## API Limits:
- **Free:** 60 API calls/minute
- **Paid:** Unlimited calls
- Perfect for trading bot usage patterns

Your trading bot will now fetch **REAL market data** from Finnhub instead of mock data!
